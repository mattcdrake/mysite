Exercise 1.9

; Version 1
(define (+ a b)
    (if (= a 0)
        b
        (inc (+ (dec a) b))))

; Version 2
(define (+ a b)
    (if (= a 0)
        b
        (+ (dec a) (inc b))))

Version 1 expands in a way that is linearly recursive. The space requirement is a factor of n where n is the number supplied as the formal parameter a.

(+ 4 5)
(inc (+ 3 5))
(inc (inc (+ 2 5)))
(inc (inc (inc (+ 1 5))))
(inc (inc (inc (inc (+ 0 5)))))
(inc (inc (inc (inc 5))))
(inc (inc (inc 6)))
(inc (inc 7))
(inc 8)
9

Version 2 is a linearly iterative process and the space requirement is constant regardless of how n scales.

(+ 4 5)
(+ 3 6)
(+ 2 7)
(+ 1 8)
(+ 0 9)
